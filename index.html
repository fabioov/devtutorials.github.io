SAP/Fiori & Javascript.

SAP Fiori is a user experience design framework developed by SAP that enhances the interaction between users and SAP applications. It uses modern design principles to create a simplified, personalized, and responsive user experience across various devices. On the other hand, JavaScript is a programming language widely used in web development. The connection? SAP Fiori applications often leverage JavaScript, specifically SAPUI5, a JavaScript-based framework, to create interactive and highly functional web apps that seamlessly integrate with SAP's suite of solutions.
